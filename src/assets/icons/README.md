## Usage

Import and use SVG icons as React components or reference via CSS classes.

```javascript
// Option 1: Import as component
import { ReactComponent as WifiIcon } from './icons/wifi.svg';

// Option 2: Use icon class (if using icon font)
<span className="icon icon--wifi"></span>
```
